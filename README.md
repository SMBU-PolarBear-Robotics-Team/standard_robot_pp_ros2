# ROS2_StandardRobot++

![Logo]()

## 介绍

ROS2 StandardRobot++ 是配合 StandardRobot++ 下位机控制使用的机器人驱动，提供了机器人的控制接口、数据接口。

- **机器人控制接口：**
  - 运动的速度向量 vx vy wz
  - 云台的朝向 pitch yaw
  - 底盘的姿态 roll pitch yaw leg_length
  - 发射机构状态 fire fric_on
